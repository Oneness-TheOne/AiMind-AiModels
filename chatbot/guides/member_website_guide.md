## 아이마음 센터 웹사이트 화면별 상세 이용 가이드 (Front 기준)

이 문서는 `Front/app` 안에 구현된 실제 화면 구조를 기준으로,  
**사용자가 어떤 화면에서 어떤 버튼·입력란·탭을 눌러야 하는지**를 최대한 구체적으로 정리한 가이드입니다.  
챗봇이 이 문서를 참고하면, 사용자가 웹사이트 이용 방법에 대해 어떤 질문을 하더라도  
**"어디에 있는 무슨 버튼을 눌러야 하는지"**를 자세히 안내할 수 있습니다.

---

### 1. 공통 레이아웃 구조 (`Header`, `Footer`, 챗봇)

- **상단 고정 헤더 (`Header`)**
    - 항상 화면 맨 위에 고정되어 있는 바입니다.
    - **왼쪽 로고**
        - 텍스트 로고: `아이마음` + `센터`
        - 클릭 시 언제나 **메인 홈(`/`)** 으로 이동합니다.
    - **가운데/오른쪽 메뉴 (데스크톱, lg 이상)**
        - 다음 메뉴 텍스트가 가로로 나열됩니다.
            - `그림 분석` → `/analysis`
            - `그림일기 OCR` → `/diary-ocr`
            - `맞춤 솔루션` → `/solutions`
            - `주변 상담소` → `/counseling`
            - `맘스퀘어` → `/community`
            - `마이페이지` → `/mypage`
        - 각 메뉴는 **텍스트 전체가 클릭 영역**입니다.
    - **오른쪽 로그인 버튼 (데스크톱)**
        - 텍스트: `로그인`
        - 클릭 시 `/login` 으로 이동합니다.
    - **모바일 메뉴 (우측 상단 햄버거 버튼)**
        - `☰` 아이콘 버튼 (원형, 투명/흰색 배경)
        - 클릭 시 오른쪽에서 **슬라이드 패널**이 열립니다.
            - 위쪽에는 `아이마음 센터` 로고 텍스트
            - 그 아래 세로 목록 메뉴:
                - `그림 분석`, `그림일기 OCR`, `맞춤 솔루션`, `주변 상담소`, `맘스퀘어`, `마이페이지`
            - 맨 아래에 **전체 폭 `로그인` 버튼**
        - 각 메뉴를 누르면 해당 페이지로 이동하며, 패널은 자동으로 닫힙니다.
    - **헤더 배경 색 변화**
        - 메인 홈(`/`) 첫 화면에서 스크롤이 거의 안 된 상태: 배경이 **투명**에 가깝고, 로고/텍스트가 배경 위에 떠 있는 느낌입니다.
        - 스크롤을 내리거나 다른 페이지에서는: **흰색 배경 + 얇은 하단 테두리 + 약한 그림자**로 표시됩니다.

- **하단 푸터 (`Footer`)**
    - 항상 페이지 맨 아래에 있습니다.
    - **좌측 영역**
        - 작은 원형 아이콘 + `아이마음` 텍스트 로고
        - 한 줄 설명:  
          `"AI 기반 아동 그림 심리 분석으로 우리 아이의 마음을 이해합니다."` (줄바꿈 포함)
    - **서비스 링크**
        - `그림 분석` → `/analysis`
        - `맞춤 솔루션` → `/solutions`
        - `주변 상담소` → `/counseling`
    - **커뮤니티 링크**
        - `맘스퀘어` → `/community`
        - `전문가 칼럼` → `/community/expert` (라우팅은 추후 구현될 수 있음)
        - `Q&A` → `/community/qna`
    - **고객지원 링크**
        - `서비스 소개` → `/about`
        - `자주 묻는 질문` → `/faq`
        - `문의하기` → `/contact`
    - **맨 아래 라인**
        - 왼쪽: `2026 아이마음. All rights reserved.`
        - 오른쪽:
            - `개인정보처리방침` → `/privacy`
            - `이용약관` → `/terms`

- **우측 하단 챗봇 플로팅 버튼 (`ChatbotModal`)**
    - 모든 페이지 오른쪽 아래에 **둥근 원형 파란 버튼**이 떠 있습니다.
    - 아이콘: 말풍선 (`MessageCircle`)
    - 버튼을 클릭하면:
        - 작은 **채팅 창**이 오른쪽 아래에 뜨고, 버튼은 작아지면서 사라집니다.
        - 채팅 창 구성:
            - 상단 파란 헤더:
                - 왼쪽: 로봇 아이콘 + `아이마음 상담 도우미` 제목, `무엇이든 물어보세요` 부제
                - 오른쪽: **최소화 아이콘**, **닫기(X)** 아이콘
            - 중앙: 대화 내용 스크롤 영역
                - 말풍선 형태로 `사용자`(오른쪽, 파란색) / `도우미`(왼쪽, 흰색)를 구분
            - 하단: 입력창
                - 텍스트 입력 `메시지를 입력하세요...`
                - 우측 `종이비행기(Send)` 아이콘 버튼
            - 제일 아래 작은 안내 문구:
                - `"AI 상담 도우미는 참고용이며, 전문 상담을 대체하지 않습니다."`
    - 최소화/닫기를 누르면 다시 플로팅 버튼만 보입니다.

### 2. 메인 홈 화면 (`/`)

홈 화면은 여러 섹션이 세로로 이어지는 랜딩 페이지 형태입니다.

- **(1) 히어로 섹션 (`HeroSection`)**
    - **배경**
        - 전체 화면을 채우는 **동영상 배경** 위에 어두운 반투명 오버레이가 덮여 있습니다.
    - **중앙 텍스트**
        - 큰 제목:
            - `아이의 마음을 이해하는 시대,`
            - 아래 줄에 `아이마음`(일부 글자는 파란색 강조)
        - 설명 문장:
            - `"아이가 말로 표현하지 못하는 감정이 있듯, 그림에는 심리 분석이 필요합니다"`  
              (중간 `심리 분석` 부분은 파란색/강조)
    - **주요 버튼**
        - 큰 흰색 버튼(둥근 모양): `빠른 상담 신청하기`
        - 클릭 시 **`/analysis` (그림 분석 시작 화면)** 으로 이동합니다.

- **(2) 서비스 특징 카드 섹션 (`FeaturesSection`)**
    - 세 개의 카드가 가로(데스크톱) / 세로(모바일) 배열로 등장합니다.
    - 각 카드 구성:
        - 상단: 그림 이미지 (아이/가족/상담 느낌 사진)
        - 텍스트:
            - 작은 설명 줄: 예) `아이의 마음을 읽는`
            - 진한 타이틀: 예) `AI 그림 분석`, `성장 추적 시스템`, `안전한 공간`
            - 간략 설명 문구 2~3줄
        - 체크 아이콘 리스트:
            - `최신 딥러닝 기반 분석`, `아동 심리 전문가 검증` 등 체크 항목 2개 정도
    - 클릭 동작은 없고, **정보 소개용** 섹션입니다.

- **(3) 통계/효과 섹션 (`StatsSection` + `HowItWorksSection`)**
    - 동그란 통계 배지들:
        - 예: `브랜드 대상 1위`, `분석 만족도 4.9/5점`, `누적 분석 수 10만 건`, `온라인 24시 운영` 등
    - 그 아래 **체크 리스트**:
        - `"아이의 숨겨진 감정을 이해하게 됩니다."`
        - `"발달 상태를 객관적으로 파악하게 됩니다."`
        - `"아이와의 소통 방법을 알게 됩니다."`
    - 이 섹션은 클릭 요소보다는 **신뢰·효과 강조** 역할입니다.

- **(4) 사용 방법 요약 섹션 (`CTASection`)**
    - 제목: `아이마음 맞춤 분석 프로세스`
    - 네 개의 STEP 아코디언(접었다 펴는 영역)이 있습니다.
        - `STEP 01` ~ `STEP 04` 까지 한 줄 제목과 간단 시간표시(예: `*1분 이내 소요`)가 옆에 보입니다.
        - 각 줄 오른쪽엔 `∨ / ∧` 아이콘이 있어 클릭 시 상세 설명이 열립니다.
        - 열렸을 때는 해당 STEP 아래에 **자세한 설명 문단**이 나타납니다.
    - 사용자는 **각 STEP 제목 줄을 클릭**해서 상세 설명을 펼쳤다가 닫을 수 있습니다.

- **(5) 앱/그림 분석 안내 섹션 (`DailyTipSection`)**
    - 상단 텍스트:
        - `"센터에 직접 연락하기 어렵다면,"`
        - `앱으로 간편하게 그림 분석`
    - 버튼:
        - 둥근 `그림 분석하기` 버튼 (클릭 시 `/analysis`)
    - 중앙 카드:
        - 왼쪽: 휴대폰 화면 모양(두 개 겹친 폰) 이미지
        - 오른쪽: 2개의 기능 카드
            - 각 카드 상단은 색이 있는 바 + 아이콘 (`Lock`, `Calendar` 등)
            - 아래에는 체크 리스트와 작은 미니 UI 프리뷰
        - 모든 요소는 **클릭 없이 읽기 전용**입니다.

### 3. 로그인 화면 (`/login`)

로그인 화면은 **상단 간단 헤더 + 중앙 카드**로 구성됩니다.

- **상단 헤더**
    - 왼쪽 상단에 `← 홈으로 돌아가기` 링크
        - 텍스트 전체가 클릭 영역이며, 클릭 시 `/` 로 이동합니다.

- **중앙 메인 영역**
    - 상단 로고
        - 원 안에 웃는 얼굴 아이콘 + `아이마음` 텍스트
        - 바로 아래에 작은 설명: `아이의 마음을 이해하는 첫 걸음`
    - 로그인 카드
        - 제목: `로그인`
        - 설명: `계정에 로그인하여 서비스를 이용하세요`
        - **입력 폼**
            - `이메일` 입력칸
                - placeholder: `example@email.com`
                - 필수 입력 (비어 있으면 제출 불가)
            - `비밀번호` 입력칸
                - placeholder: `비밀번호를 입력하세요`
                - 오른쪽 끝에 **눈 모양 아이콘 버튼**:
                    - 클릭 시 `●●●` → 일반 텍스트로 토글
                    - 다시 클릭하면 다시 숨겨짐
                - `비밀번호 표시 토글`은 시각장애인을 위한 숨김 텍스트도 포함됨.
        - **하단 옵션/링크**
            - 왼쪽: 체크박스 + `로그인 상태 유지`
                - 클릭 시 체크/해제
            - 오른쪽: `비밀번호 찾기` 텍스트 링크
                - 현재는 `/forgot-password` 로 연결(추후 실제 페이지 구현 가능)
        - **로그인 버튼**
            - 텍스트: `로그인`
            - 클릭 시 1초 정도 로딩 후, **임시로 `/mypage` 로 이동**하는 동작(실제 백엔드 연동 전).
    - **소셜 로그인 영역**
        - 구분선 위에 `또는`
        - 버튼 2개:
            - `Google로 계속하기`
            - `카카오로 계속하기`
        - 현재는 디자인/레이아웃만 구현된 상태로, 실제 인증은 나중에 연동될 수 있습니다.
    - **회원가입 유도 문구**
        - `"아직 계정이 없으신가요? 회원가입"`
        - `회원가입` 링크 클릭 시 `/signup` 으로 이동.

### 4. 회원가입 화면 (`/signup`)

회원가입 화면은 로그인과 유사한 구조지만, **입력 항목과 비밀번호 조건 안내**가 더 많습니다.

- **상단 헤더**
    - `← 홈으로 돌아가기` 링크 (클릭 시 `/`)

- **로고/설명**
    - `아이마음` 로고 + `아이의 마음을 이해하는 첫 걸음`

- **회원가입 카드**
    - 제목: `회원가입`
    - 설명: `계정을 만들어 아이마음 서비스를 시작하세요`

- **입력 항목**
    - `이름`
        - placeholder: `홍길동`
    - `이메일`
        - placeholder: `example@email.com`
    - `비밀번호`
        - 오른쪽 눈 아이콘으로 보이기/숨기기 토글
        - 아래에 **비밀번호 조건 3개가 실시간 표시**:
            - `8자 이상`
            - `영문 포함`
            - `숫자 포함`
        - 각 조건은 충족되면 동그라미 안에 체크아이콘 + 색상이 바뀜.
    - `비밀번호 확인`
        - 위 비밀번호와 다르면 아래에 **빨간색 경고 문구**: `비밀번호가 일치하지 않습니다`
        - 오른쪽 눈 아이콘으로 토글 가능.
    - `자녀 연령대` 선택
        - 드롭다운(Select) 클릭 시 다음 중 하나를 고를 수 있습니다.
            - `3-4세`, `5-6세`, `7-8세`, `9-10세`, `11-12세`, `13세 이상`

- **약관 동의 영역**
    - 체크박스 항목 3개:
        - `[필수] 이용약관에 동의합니다` (링크: `/terms`)
        - `[필수] 개인정보처리방침에 동의합니다` (링크: `/privacy`)
        - `[선택] 마케팅 정보 수신에 동의합니다`
    - **필수 2개(이용약관, 개인정보) 체크 전에는 `회원가입` 버튼이 비활성화** 됩니다.

- **하단 버튼**
    - `회원가입` 버튼
        - 필수항목/비밀번호 일치 조건 충족 시 활성화
        - 클릭 시 1초 정도 딜레이 후 `/login` 으로 이동
    - 구분선 아래 소셜 버튼 2개:
        - `Google로 계속하기`, `카카오로 계속하기` (레이아웃 동일, 실제 인증 추후 연동)
    - 맨 아래 문구:
        - `"이미 계정이 있으신가요? 로그인"`
        - `로그인` 링크 클릭 시 `/login`

### 5. 그림 분석 메인 (`/analysis`)

그림 분석 메인 화면은 **동영상형 헤더 + 그림 입력 카드 + 아이 정보 카드 + 간단 가이드**로 나뉩니다.

- **상단 비디오 히어로 (`VideoHero`)**
    - 소제목: `AI 기반 심리 분석`
    - 큰 제목: `그림으로 읽는 아이의 마음`
    - 설명: `"아이의 그림을 업로드하고 정보를 입력하면 AI가 심리 분석을 진행합니다."`

- **(1) 그림 입력 카드**
    - 카드 제목: `그림 입력`
    - 설명: `그림을 업로드하거나 직접 그려주세요`
    - 상단 탭(`Tabs`) 두 개:
        - `업로드` 탭:
            - **아직 업로드 전**
                - 점선 테두리 큰 영역
                - 중앙 아이콘: `Upload`
                - 텍스트:
                    - `이미지를 여기에 드래그하세요`
                    - `또는 클릭하여 파일을 선택하세요`
                - 하단에 `파일 선택` 버튼:
                    - 클릭 시 파일 선택창 열림 (이미지 파일만)
                - 지원 형식 안내:
                    - `JPG, PNG, HEIC (최대 10MB)`
                - **사용법**
                    - 파일을 **드래그 앤 드롭**하거나, `파일 선택` 버튼을 눌러 선택합니다.
            - **이미지를 업로드한 경우**
                - 업로드한 그림이 카드 안에 크게 미리보기로 표시됩니다.
                - 오른쪽 상단 붉은 `X` 아이콘 버튼:
                    - 클릭 시 업로드한 이미지를 제거하고 다시 초기 상태로 돌아갑니다.
        - `직접 그리기` 탭:
            - 하얀 캔버스 영역(그리기 도구 포함)이 노출됩니다.
            - 그림을 그린 뒤, 캔버스 안의 `저장` 버튼(컴포넌트 내부)에 따라  
              **저장 시 자동으로 업로드 탭으로 이미지가 이동**합니다.

- **(2) 아이 정보 입력 카드**
    - 카드 제목: `아이 정보 입력`
    - 설명: `정확한 분석을 위해 아이의 정보를 입력해주세요`
    - 입력 필드:
        - `아이 이름 (별명)` (텍스트)
        - `그림 유형` (드롭다운)
            - 선택 옵션:
                - `집-나무-사람 (HTP)`
                - `가족화`
                - `동적 가족화 (KFD)`
                - `자유화`
                - `인물화`
        - `나이` (드롭다운)
            - `3세` ~ `15세` 까지 선택 가능 (3부터 15까지 숫자)
        - `성별` (드롭다운)
            - `남아`, `여아`
    - **분석 시작 버튼**
        - 텍스트: `분석 시작하기`
        - 오른쪽에 화살표 아이콘
        - 버튼 활성 조건:
            - **업로드된 이미지가 있어야 하고**,
            - `나이`, `그림 유형` 이 선택되어 있어야 합니다.
        - 클릭 시 `/analysis/analyzing` 페이지로 이동합니다.

- **(3) 하단 분석 가이드 카드**
    - 제목: `그림 분석 가이드`
    - 번호 1~3 까지 작은 원 안 숫자 표시 + 짧은 텍스트:
        - `선명한 이미지` / `전체 그림 포함` / `정확한 정보`
    - **실제 버튼은 없고, 읽기용 안내**입니다.

### 6. 분석 진행 화면 (`/analysis/analyzing`)

이 화면은 **진행 중 애니메이션 + 꿀팁** 역할을 합니다.

- 중앙 카드 안 구성:
    - 로고 원 + `그림을 분석하고 있어요` 제목
    - 작은 설명: `AI가 아이의 그림을 세심하게 분석 중입니다`
    - **진행률 바**
        - `분석 진행률` + 우측에 `0% ~ 100%` 숫자
        - 하단에 굵은 프로그래스 바
    - **단계 리스트 네 개**
        - `구성요소 분석 중...`
        - `발달 단계 평가 중...`
        - `또래 비교 중...`
        - `종합 분석 중...`
        - 현재 단계는 배경/아이콘 색이 더 진하게 표시되고,  
          오른쪽에 작은 깜빡이는 점(애니메이션)이 나타납니다.
    - **하단 팁 박스**
        - 전구 아이콘 + `알고 계셨나요?`
        - 3초마다 다른 문장이 자동으로 바뀌며 표시됩니다.
            - 예: `아이들은 그림을 통해 언어로 표현하기 어려운 감정을 나타내요.` 등
    - 진행률이 100%에 도달하면 잠시 후 **자동으로 `/analysis/result` 로 이동**합니다.

### 7. 분석 결과 화면 (`/analysis/result`)

분석 결과 화면은 여러 **탭**으로 구성된 리포트 페이지입니다.

- **상단 헤더 영역**
    - 좌측:
        - 상단 배지: `분석 완료`
        - 날짜: 오늘 날짜(로컬 기준)
        - 큰 제목: `[아이이름]의 그림 분석 결과` (예: `민준이의 그림 분석 결과`)
        - 부제: `7세 | 집-나무-사람 (HTP)` 처럼 나이와 그림 유형
    - 우측:
        - `공유` 버튼 (아이콘 + 텍스트)
        - `PDF 저장` 버튼
        - 현재는 UI만, 실제 공유/PDF 저장은 이후 연동 가능.

- **종합 요약 카드**
    - 왼쪽:
        - 큰 네모 안에 **숫자 점수** (예: `85`)
        - 아래 텍스트: `종합 점수`, `상위 15%`
    - 가운데:
        - 한 문장 요약: `"전반적으로 안정적인 정서 상태..."` 등의 해석
    - 오른쪽:
        - `발달 단계`: 예) `정상 발달`
        - `정서 상태`: 예) `안정`

- **탭 구조**
    - 상단 탭 4개:
        - `기본 분석`
        - `발달 비교`
        - `심리 해석`
        - `추천 사항`
    - 각 탭을 클릭하면 아래 내용이 해당 탭으로 바뀝니다.

- **(1) 기본 분석 탭**
    - 왼쪽 카드: 그림 개요
        - 단순화된 집-나무-사람 그림 그림(아이콘 형식)
        - 아래에 `집`, `나무`, `사람`, `태양` 이 어떤 색으로 강조되었는지 범례(배지) 표시
    - 오른쪽 카드: 구성요소 리스트
        - 예: `집`, `나무`, `사람`, `태양`, `구름`, `꽃/풀`
        - 각 요소마다:
            - 왼쪽 아이콘(집, 나무, 사람, 태양 이모지 등)
            - `감지됨` 또는 `미감지` 배지
            - 작은 설명 문장: `"안정적인 형태, 굴뚝에서 연기 표현"` 등

- **(2) 발달 비교 탭**
    - 왼쪽: 세로 막대 그래프 (`또래 비교`)
        - 항목: `세부묘사`, `색상사용`, `공간활용`, `비율표현`, `창의성`
        - 막대 두 개씩: 내 아이 점수 vs 또래 평균
    - 오른쪽: 발달 단계 평가 카드
        - 현재 발달 단계 표시: 예) `도식기 (7-9세)`
        - 밑에 4개의 프로그레스 바:
            - `그림 복잡도`, `세부 표현력`, `공간 인식`, `비율 표현` 각각 현재 점수 표시

- **(3) 심리 해석 탭**
    - 왼쪽: 레이더 차트(`심리 지표`)
        - 축: `자아 존중감`, `정서 안정`, `사회성`, `창의성`, `가족 관계`
    - 오른쪽: 색상 심리 분석
        - `파랑`, `초록`, `노랑`, `빨강`, `갈색` 각각
            - 색상 박스 + 퍼센트 + 의미 텍스트 + 진행바
    - 하단: 감정 상태 해석 카드들
        - `자아 존중감`, `정서 안정`, `사회성`, `창의성`, `가족 관계` 각각
            - 점수 배지, 진행바, `"양호/보통/관심 필요"` 라벨

- **(4) 추천 사항 탭**
    - 3개의 카드:
        - `가정 활동`, `미술 활동`, `정서 발달` 카테고리
        - 각 카드 안에 1~3번으로 된 구체적인 추천 문장 리스트
    - 하단 CTA 카드:
        - `"더 자세한 맞춤 솔루션이 필요하신가요?"`
        - `맞춤 솔루션 보기` 버튼 클릭 시 `/solutions` 로 이동.

### 8. 그림일기 OCR 화면 (`/diary-ocr`)

이 화면은 **기능 소개 + OCR 컴포넌트 + 촬영 팁**으로 구성됩니다.

- 상단 비디오 히어로
    - 소제목: `AI 손글씨 인식`
    - 제목: `그림일기 OCR`
    - 설명: `"그림일기 손글씨를 AI가 텍스트로 변환해드립니다..."` (요약)

- 특징 카드 4개
    - `AI 손글씨 인식`, `그림일기 보관`, `빠른 변환`, `안전한 보관`  
      각각 아이콘 + 짧은 설명.

- 중앙 메인 OCR 영역 (`DiaryOCR` – Front 전용 버전)
    - **업로드 전 상태**
        - 큰 점선 박스
        - 중앙 아이콘 `Upload` +  
          `"그림일기 이미지를 드래그하세요"` / `"또는 클릭하여 파일을 선택하세요"`
        - `파일 선택` 버튼 (이미지 파일)
        - 지원 형식 안내 (`JPG, PNG, HEIC (최대 10MB)`)
    - **이미지 업로드 후**
        - 왼쪽: 업로드한 그림일기 이미지 미리보기 + 오른쪽 상단 `X` 삭제 버튼
        - 아래 `텍스트 추출하기` 버튼
            - 클릭 시 `/api/ocr` 호출 → 추출 결과를 오른쪽 텍스트 영역에 표시
        - 오른쪽: `추출된 텍스트` 영역
            - 큰 텍스트 박스 (직접 수정 가능)
            - 상단 `복사` 버튼: 텍스트 전체 클립보드로 복사
            - 하단 버튼 2개:
                - `다시 추출` (같은 이미지로 재시도)
                - `저장하기` (현재 텍스트와 이미지를 saved 리스트에 추가)
    - **하단 "저장된 그림일기" 리스트**
        - 작은 썸네일 이미지 + 아이 이름 + 날짜 + 텍스트 요약
        - 오른쪽에 `복사` 아이콘 버튼 (각 항목별 텍스트 복사)
        - 항목을 클릭했을 때는 현재는 별도 상세 페이지 없이 스타일만 변경됨.

- 하단 촬영 팁 섹션
    - `더 정확한 인식을 위한 팁` 제목
    - 1~4번 카드:
        - `밝은 곳에서 촬영하세요`, `그림자를 피해주세요`, `수평을 맞춰주세요`, `선명하게 찍어주세요`

        - placeholder: `궁금한 내용을 검색해보세요`

        - 입력 시 게시글 리스트가 실시간 필터링됩니다.

        - 각 태그 버튼을 누르면 검색창에 해당 태그 문자열이 자동으로 채워집니다.

    - 4개의 숫자 카드:
        - `회원`, `게시글`, `댓글`, `전문가` 수를 보여줍니다.

- **메인 콘텐츠 – 게시글 리스트**
    - 상단 카테고리 버튼들 (왼쪽):
        - `전체`, `자유게시판`, `육아 꿀팁`, `Q&A`, `상담 후기`, `전문가 칼럼`

    - 정렬 옵션 버튼들:
        - `인기순`, `최신순`
    - 게시글 카드(리스트 형태, 각 행 클릭 가능):
        - 상단에 카테고리 라벨 + (있다면) 글쓴이 뱃지(`전문가`, `인기 작성자` 등)
        - 제목 (한 줄, hover 시 파란색)
        - 내용 요약 (한 줄)
        - 하단 왼쪽:
            - 작성자 이름 + 작성 시간 (예: `10분 전`)
        - 하단 오른쪽:
            - `좋아요` 하트 아이콘 + 숫자
                - 아이콘 클릭 시 색상/숫자가 토글(증가/감소)
            - `댓글` 아이콘 + 개수
            - `조회수` 아이콘 + 숫자
            - `북마크` 아이콘 (저장 표시) 토글
        - 제목 또는 카드 아무 곳을 클릭하면 `/community/[id]` 상세 페이지로 이동.
    - 리스트 하단:
        - 페이지네이션(`이전/다음` 버튼 + 페이지 번호)

- **모바일 글쓰기 버튼**
    - 화면 맨 아래 쪽에 `글쓰기` 전체폭 버튼이 있습니다.
    - 클릭 시 `/community/write` 이동.

- **오른쪽 사이드바**
    - `활동 중인 전문가` 리스트:
        - 전문가 이름, 직함(예: `아동심리상담사`), 답변 수 (`답변 234`)
    - `인기 태그` 버튼들:
        - `#그림분석`, `#발달검사` 등 (클릭 시 검색창에 태그 반영)
    - 하단 배너:
        - `전문가 상담 받기` 제목
        - `상담 신청` 버튼 → `/counseling` 이동

        - 멀티라인 텍스트 박스
        - placeholder: `내용을 입력하세요 (최소 10자)`
        - 우측 하단 글자 수 카운트 (`0/5000`)
    - `이미지 첨부 (최대 5장)`
        - 현재 업로드된 이미지는 작게 썸네일 + 오른쪽 상단 `X` 제거 버튼
        - 빈 칸(점선 박스) 클릭 시 파일 선택창, 최대 5장까지
    - `태그 (최대 5개)`
        - 태그 입력창
            - `Enter` 또는 쉼표(,) 입력 시 태그 추가
            - 앞에 `#`를 붙이면 자동으로 제거 후 태그 텍스트만 저장
        - 이미 추가된 태그는 작은 배지로 표시되며, `X` 버튼으로 제거 가능
        - 아래에 `추천:` 으로 시작하는 추천 태그 목록을 클릭하면 자동 추가됩니다.
        - 태그 개수가 5개 이상이면 더 이상 추가/입력 불가.
    - 하단 `커뮤니티 이용 수칙` 박스
        - 점 목록으로 **예의, 개인정보, 광고글 제한, 아이 사진 주의** 등을 안내.

- 하단 버튼
    - 왼쪽: `취소` 버튼 → `/community` 로 이동
    - 오른쪽: `등록하기` 버튼
        - 카테고리 + 제목 5자 이상 + 내용 10자 이상일 때만 활성화
        - 클릭 시 (현재 구현 기준) 백엔드 저장 없이 `/community` 로 이동.

### 12. 주변 상담소 찾기 (`/counseling`)

이 화면은 **검색/필터 헤더 + 지도/리스트 전환 + 센터 카드**로 구성됩니다.

- **상단 검색 헤더**
    - 검색창
        - placeholder: `상담센터 이름 또는 주소 검색`
        - 입력 시 센터 목록 필터링
    - `필터` 버튼
        - 클릭 시 오른쪽에서 필터 설정 패널(Sheet)이 열립니다.
        - `전문 분야` 체크박스 목록:
            - `놀이치료`, `미술치료`, `언어치료`, `발달검사`, `가족치료`, `부모상담`, `ADHD`, `감각통합`, `인지치료`
        - `필터 초기화` 버튼: 선택된 전문 분야 전부 해제
    - 정렬 드롭다운
        - `거리순`, `평점순` 중 선택
    - **지도/리스트 전환 토글**
        - 작은 두 개의 아이콘 버튼:
            - 지도 아이콘: `map` 모드
            - 리스트 아이콘: `list` 모드
        - 각각 클릭 시 **현재 보기 모드가 전환**됩니다.

- **지도 모드(viewMode = "map")**
    - 왼쪽 큰 지도 영역
        - 카카오 지도 SDK 로드 후, 각 상담소 위치에 마커 표시
        - 마커 클릭 시 오른쪽 센터 카드에서 해당 센터가 선택 상태가 됩니다.
    - 오른쪽 센터 리스트
        - 상단에 `"X개의 상담센터"` 텍스트
        - 아래로 스크롤 가능한 카드 리스트:
            - 각 카드(센터)의 구성:
                - 상단: 센터 이름 + `영업중`/`영업종료` 배지
                - 주소 (앞에 핀 아이콘)
                - 평점/리뷰 수 (`⭐ 4.8 (124)`), 거리(예: `0.5km`)
                - 영업 시간 (`09:00 - 18:00`)
                - 전문 분야 배지들 (`놀이치료`, `미술치료` 등)
            - 카드 전체가 클릭 영역이며, 클릭 시 **선택 상태(외곽선 강조)** 와 함께  
              모바일에서는 하단 요약 패널에 정보가 표시됩니다.

- **리스트 모드(viewMode = "list")**
    - 지도 없이 **센터 카드들만** 그리드(1~3열)로 나열됩니다.
    - 카드 구조/동작은 지도 모드의 센터 카드와 동일합니다.

- **모바일 하단 선택 센터 패널**
    - 작은 화면에서 특정 센터를 선택하면 화면 맨 아래에 고정 패널이 뜹니다.
    - 내용:
        - 센터 이름 + `영업중/영업종료` 배지
        - 주소, 평점(별점), 거리
        - 오른쪽에 `전화` 버튼:
            - 모바일에서는 클릭 시 해당 번호로 통화 연결 시도 (기기 기본 동작)

### 13. 맞춤 솔루션 (`/solutions`)

이 화면은 분석 결과를 바탕으로 **치료법/색상/가정 활동**을 제안하는 페이지입니다.

- 상단 비디오 히어로
    - 소제목: `전문가 추천`
    - 제목: `맞춤 솔루션`

- 상단 탭
    - `치료법 제안`
    - `색상 추천`
    - `활동 가이드`

- **(1) 치료법 제안 탭**
    - 위쪽 카드 3개 (선택 가능):
        - `미술 치료`, `놀이 치료`, `독서 치료`
        - 각 카드:
            - 왼쪽: 색상 있는 아이콘 박스
            - 오른쪽: 제목 + 짧은 설명
        - 카드 전체 클릭 시 **해당 치료법이 선택**되고, 아래 활동 리스트가 바뀝니다.
    - 하단 활동 아코디언:
        - 선택된 치료법에 따라 2~3개 `활동`이 표시됩니다.
        - 각 활동은:
            - 제목, 예상 소요 시간 (예: `20분`)
            - 설명 문단
            - `진행 방법` 리스트 (번호 대신 오른쪽 화살표 아이콘이 앞에 붙은 글머리)

- **(2) 색상 추천 탭**
    - 네 개의 카드:
        - `불안함`, `우울함`, `분노`, `외로움` 등 감정 상태별
        - 각 카드:
            - 상단: `"불안함을 느낄 때"` 같은 제목
            - 우측 상단: 추천 색상 동그란 칩 2개 (예: 파랑, 초록)
            - 설명: `"차분하고 안정적인 색으로 마음 진정"`
            - 아래 `추천 색상: 파랑, 초록`
            - `추천 활동` 리스트 (예: `파란 하늘 그리기`, `숲 풍경 색칠하기` 등)
    - 하단 팁 카드:
        - `"색상 활용 팁"` 제목
        - 색 선택 강요 금지, 선호 색의 변화는 자연스러움 등 설명.

- **(3) 활동 가이드 탭**
    - 2열 카드 목록:
        - 예: `그림 일기 쓰기`, `가족 미술 시간`, `감정 카드 게임`, `자연물 아트`
        - 각 카드:
            - 제목 + 권장 연령대 배지 (예: `5-12세`)
            - 설명, 소요 시간, 주당 빈도
            - `준비물` 배지 리스트
            - `기대 효과` 글머리 리스트
    - 하단 CTA 카드:
        - `"전문가 상담이 필요하신가요?"` 문구
        - `주변 상담소 찾기` 버튼 → `/counseling`

### 14. 마이페이지 메인 (`/mypage`)

마이페이지는 **상단 프로필 배너 + 통계 카드 3개 + 탭 구조**로 구성됩니다.

- **상단 프로필 배너**
    - 배경: 초록/청록 계열 그라데이션
    - 좌측: 큰 원형 아바타 (`김` 글자)
    - 텍스트:
        - 이름: `김미래`
        - 이메일: `miracle.mom@email.com`
        - 가입일, 총 분석 횟수 (예: `분석 4회`)
    - 오른쪽: `설정` 버튼 → `/mypage/settings`

- **요약 통계 카드 3개**
    - `총 분석`(개수)
    - `평균 점수`(예: 80점)
    - `등록 아이`(예: 2명)

- **탭 목록**
    - `개요`, `분석 기록`, `그림일기 OCR`, `아이 관리`, `활동`, `저장됨`

- **(1) 개요 탭**
    - 발달 영역별 분석 카드 4개:
        - `정서 발달`, `사회성 발달`, `인지 발달`, `창의성`
        - 각 카드에 현재 점수 + 이전 대비 상승/하락 수치
    - `최근 분석 기록`
        - 최근 3개 분석 이력 카드:
            - 그림 유형, 아이 이름/나이, 날짜, 점수 등
            - 클릭 시 현재는 시각적 효과만, 추후 상세로 연결 가능.

- **(2) 분석 기록 탭**
    - 상단:
        - 제목 `분석 기록`
        - `"새 분석"` 버튼 → `/analysis`
    - 아래 리스트:
        - 페이징된 분석 이력 카드 (위 개요 탭 카드와 비슷하나, 오른쪽에 PDF/공유 아이콘 버튼 포함)
        - 하단 페이지네이션으로 이전/다음 페이지 이동.

- **(3) 그림일기 OCR 탭**
    - `DiaryOCR` (마이페이지 버전) 컴포넌트가 그대로 출력됩니다.
    - 구조/사용법은 `/diary-ocr` 페이지의 OCR 영역과 거의 동일하나,
        - 마이페이지 스타일에 맞게 애니메이션/페이지네이션이 조금 다릅니다.

- **(4) 아이 관리 탭**
    - 상단:
        - `"아이 관리"` 제목
        - `아이 추가` 버튼 (추후 폼/모달과 연동 가능)
    - 아래 `아이 카드` 목록:
        - 각각:
            - 원형 배경(색상) + 아이 첫 글자, 이름, 나이, 성별
            - 하단: 생년월일, 분석 횟수
        - 카드 전체 클릭 시 추후 상세 페이지와 연동 가능(현재는 시각적 효과만 있음).

- **(5) 활동 탭**
    - 커뮤니티 관련 활동 기록:
        - `post`(게시글), `comment`(댓글), `like`(좋아요) 아이콘 색상으로 구분
        - 제목, 카테고리, 날짜 표시
        - 오른쪽에는 링크 아이콘 (상세 페이지로 이동하는 의미)

- **(6) 저장됨 탭**
    - 북마크했던 게시글 목록:
        - 제목, 작성자, 카테고리, 날짜 표시
        - 오른쪽 링크 아이콘

### 15. 마이페이지 > 설정 (`/mypage/settings`)

            - `취소`, `삭제하기` 버튼

### 16. 요약: 챗봇이 자주 받을 수 있는 질문 유형 예시

아래는 이 가이드를 바탕으로 챗봇이 답변할 수 있는 질문 예시와,  
**어느 화면·어느 요소를 안내하면 되는지**를 정리한 참고 목록입니다.

- **"그림 분석은 어디서 시작하나요?"**
    - 답변 방향:
        - 상단 헤더의 `그림 분석` 메뉴 또는 홈 화면의 `빠른 상담 신청하기` 버튼을 눌러 `/analysis` 로 이동하도록 안내.
        - `/analysis` 화면에서 `업로드` 탭에서 그림 파일을 올리거나 `직접 그리기` 탭을 사용하라고 설명.

- **"그림 일기 사진을 텍스트로 바꾸고 싶어요."**
    - 답변 방향:
        - 상단 메뉴 `그림일기 OCR` 또는 URL `/diary-ocr` 로 들어가  
          큰 박스에 사진을 드래그하거나 `파일 선택` 버튼을 누르라고 안내.
        - 업로드 후 `텍스트 추출하기` 버튼을 눌러 결과를 보고, 필요 시 `복사`와 `저장하기` 기능도 알려줍니다.

- **"주변 상담센터를 어떻게 찾아요?"**
    - 답변 방향:
        - 상단 메뉴 `주변 상담소` 클릭 → `/counseling`.
        - 상단 검색창에 지역/센터 이름을 입력하고, `필터`에서 전문 분야를 선택하는 방법 안내.
        - 지도/리스트 토글 버튼을 이용해 원하는 방식으로 보는 법, 센터 카드를 눌러 상세 정보/전화 버튼을 확인하라고 유도.

- **"맘스퀘어에서 글을 쓰고 싶어요."**
    - 답변 방향:
        - `/community` 에서 오른쪽 상단 `글쓰기` 버튼(PC) 또는 하단 고정 `글쓰기` 버튼(모바일)을 누르면 `/community/write`로 이동.
        - `카테고리`, `제목`, `내용`, `이미지`, `태그` 입력 규칙(최소 글자 수, 최대 이미지 5장, 태그 5개)을 설명.

- **"내 분석 기록은 어디서 보나요?"**
    - 답변 방향:
        - 상단 메뉴 `마이페이지` → 기본 `개요` 탭에서 최근 분석 3개를 카드로 확인 가능.
        - 더 자세한 목록은 `분석 기록` 탭으로 이동하여 페이지네이션을 넘기며 확인하도록 안내.

- **"로그인/회원가입이 잘 안 보여요."**
    - 답변 방향:
        - 데스크톱: 상단 우측 `로그인` 버튼으로 `/login` 이동.
        - 로그인 화면 하단의 `회원가입` 링크로 `/signup` 이동.
        - 모바일: 우측 상단 햄버거 메뉴 → 슬라이드 패널 최하단의 `로그인` 버튼 안내.

이 문서만으로도 **각 화면의 구조와 요소 이름, 이동 경로, 버튼 활성 조건**을 모두 설명할 수 있도록 작성되어 있으므로,  
챗봇은 사용자의 질문을 받은 뒤 **"지금 위치(페이지) → 이동할 페이지 → 눌러야 하는 버튼/메뉴/입력칸"** 을 단계적으로 안내해 주면 됩니다.
