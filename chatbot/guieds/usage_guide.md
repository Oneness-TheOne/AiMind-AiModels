## 아이마음센터 이용 가이드 (FAQ / RAG 문서)

- **대상**: 아이마음센터(Front) 웹사이트 사용 방법 안내
- **범위**: `Front/app/*/page.tsx`, 공통 헤더 메뉴(`components/layout/header.tsx`) 기준
- **참고**: 일부 버튼/기능은 현재 코드상 **UI만 있고 실제 동작(서버 연동)이 없는 상태**일 수 있습니다.

## 계정/로그인

### Q. 회원가입은 어떻게 하나요?

- **관련 화면**: `/signup`
- **키워드**: 회원가입, 이용약관, 개인정보처리방침, 비밀번호 조건
- **A**:
    - 이름, 이메일, 비밀번호, 비밀번호 확인을 입력합니다.
    - 비밀번호는 **8자 이상 + 영문 포함 + 숫자 포함** 조건 충족 여부가 표시됩니다.
    - **필수 동의**: 이용약관 동의, 개인정보처리방침 동의(둘 다 체크 필요)
    - 완료 후 **로그인 화면(`/login`)으로 이동**합니다.
- **주의**:
    - 약관 링크 **`/terms`**, **`/privacy`**는 현재 `app/`에 페이지가 없어 **404가 날 수 있습니다.**

### Q. 로그인은 어떻게 하나요?

- **관련 화면**: `/login`
- **키워드**: 로그인, 이메일, 비밀번호, 로그인 상태 유지
- **A**:
    - 이메일/비밀번호를 입력하고 **로그인**을 누릅니다.
    - 현재 구현은 시뮬레이션으로, 로그인 성공 시 **마이페이지(`/mypage`)로 이동**합니다.
- **주의**:
    - **비밀번호 찾기** 링크 **`/forgot-password`**는 현재 라우트가 없어 **404가 날 수 있습니다.**
    - Google/카카오 버튼은 **UI만 존재**할 수 있습니다(실제 OAuth 연동 전).

## 그림 분석

### Q. 그림 분석은 어디서 시작하나요?

- **관련 화면**: `/analysis`
- **키워드**: 그림 분석, 상담 문의, 분석 시작하기
- **A**:
    - 상단 메뉴에서 **그림 분석**을 누르거나,
    - 헤더의 **상담 문의** 버튼을 눌러 `/analysis`로 이동합니다.

### Q. 그림을 업로드해서 분석하려면 어떻게 하나요?

- **관련 화면**: `/analysis`
- **키워드**: 업로드, 파일 선택, 드래그앤드롭, 미리보기
- **A**:
    - `업로드` 탭에서 이미지 파일을 **드래그&드롭**하거나 **파일 선택**으로 업로드합니다.
    - 업로드 후 미리보기를 확인하고, 필요하면 **X 버튼**으로 삭제 후 다시 올립니다.
    - 아래 **아이 정보**에서 필수 항목을 채운 뒤 **분석 시작하기**를 누릅니다.
- **팁(정확도)**:
    - 그림이 **선명하게**, **전체가 프레임에 들어오게** 촬영한 이미지를 권장합니다.

### Q. 웹에서 직접 그림을 그려서 분석할 수 있나요?

- **관련 화면**: `/analysis`
- **키워드**: 직접 그리기, 캔버스, 저장
- **A**:
    - `직접 그리기` 탭에서 그림을 그린 뒤 **저장**하면,
    - 저장된 이미지가 업로드 이미지로 반영되어 분석에 사용할 수 있습니다.

### Q. ‘분석 시작하기’ 버튼이 눌리지 않아요(비활성화).

- **관련 화면**: `/analysis`
- **키워드**: 분석 시작하기, 버튼 비활성화, 나이, 그림 유형
- **원인/해결**:
    - **그림이 없음** → 이미지를 업로드하거나 직접 그린 뒤 저장하세요.
    - **필수 정보 미입력** → 아래 항목을 채워야 활성화됩니다.
        - **나이(필수)**
        - **그림 유형(필수)**
- **참고**:
    - 아이 이름/성별은 코드상 **선택 입력**입니다.

### Q. 분석을 시작하면 어떤 화면이 나오고, 얼마나 걸리나요?

- **관련 화면**: `/analysis/analyzing`
- **키워드**: 분석 진행률, 로딩, 자동 이동
- **A**:
    - 분석 진행 화면에서 **진행률(%)**과 **단계(4단계)**가 표시됩니다.
    - 사용자는 별도 조작 없이 기다리면 되고, 완료 시 **자동으로 결과(`/analysis/result`)로 이동**합니다.

### Q. 분석 결과는 어디서 보고, 어떻게 읽나요?

- **관련 화면**: `/analysis/result`
- **키워드**: 기본 분석, 발달 비교, 심리 해석, 추천 사항, 탭
- **A**:
    - 상단에서 종합 요약(점수/발달/정서)을 확인합니다.
    - 아래 탭으로 상세를 확인합니다.
        - **기본 분석**: 감지된 요소(집/나무/사람 등) 중심
        - **발달 비교**: 또래 평균 비교, 발달 단계 평가
        - **심리 해석**: 심리 지표, 색상 분석, 감정 해석
        - **추천 사항**: 가정/미술/정서 발달 추천 활동
    - 추천 사항 탭에서 **맞춤 솔루션 보기**로 `/solutions` 이동이 가능합니다.
- **주의**:
    - 상단의 **공유**, **PDF 저장** 버튼은 현재 코드상 **UI만 있을 수 있습니다.**

## 그림일기 OCR

### Q. 그림일기 OCR은 어떻게 쓰나요?

- **관련 화면**: `/diary-ocr` (마이페이지 탭에서도 동일 컴포넌트 사용)
- **키워드**: OCR, 손글씨, 텍스트 추출하기, 저장하기, 복사
- **A**:
    - 이미지 업로드(드래그/파일 선택)
    - **텍스트 추출하기** 버튼 클릭 → 손글씨를 텍스트로 변환
    - 결과 텍스트는 수정 가능하며, **복사**하거나 **저장하기**로 목록에 보관할 수 있습니다.

### Q. OCR이 실패했어요. 어떻게 해야 하나요?

- **관련 화면**: `/diary-ocr`
- **키워드**: 텍스트 추출 실패, 다시 추출, 오류
- **A(체크리스트)**:
    - 이미지가 **어둡거나 흔들리면** 인식률이 떨어집니다 → 밝고 선명하게 재촬영
    - 글씨가 너무 작으면 인식이 어려울 수 있습니다 → 글씨 영역이 잘 보이게 촬영
    - 업로드 후 **다시 추출**로 재시도

### Q. OCR은 어떤 방식으로 동작하나요?

- **관련 API**: `POST /api/ocr`
- **키워드**: API, dataURL, text
- **A**:
    - 프론트에서 이미지(dataURL)를 서버로 보내고, 서버가 AI 모델을 통해 텍스트를 추출합니다.
    - 응답의 `text`를 화면에 표시합니다.

## 맞춤 솔루션

### Q. 맞춤 솔루션에서는 무엇을 할 수 있나요?

- **관련 화면**: `/solutions`
- **키워드**: 치료법 제안, 색상 추천, 활동 가이드
- **A**:
    - 탭 3개로 구성됩니다.
        - **치료법 제안**: 치료법 카드 선택 → 해당 활동 아코디언에서 진행 방법 확인
        - **색상 추천**: 감정별 추천 색상/활동 확인, 색상 활용 팁
        - **활동 가이드**: 준비물/기대효과 포함한 가정 활동 카드
    - 활동 가이드에는 **주변 상담소 찾기** 버튼이 있어 `/counseling`으로 이동합니다.

## 주변 상담소

### Q. 상담소는 어떻게 검색하나요?

- **관련 화면**: `/counseling`
- **키워드**: 상담센터 검색, 주소 검색, 필터, 거리순, 평점순
- **A**:
    - 상단 검색창에 **센터 이름 또는 주소**를 입력합니다.
    - **필터**에서 전문 분야를 체크(복수 선택 가능)합니다.
    - 정렬을 **거리순/평점순**으로 바꿀 수 있습니다.

### Q. 지도 보기/리스트 보기 전환은 어떻게 하나요?

- **관련 화면**: `/counseling`
- **키워드**: 지도, 리스트, 보기 전환
- **A**:
    - 오른쪽에 있는 아이콘 버튼으로 **지도/리스트** 모드를 전환합니다.
    - 지도 모드에서는 마커 클릭 시 해당 센터가 선택됩니다.

### Q. 지도(카카오맵)가 안 떠요.

- **관련 화면**: `/counseling`
- **키워드**: 카카오맵, API 키, 환경변수
- **A**:
    - `/api/kakao-map-key`에서 카카오맵 키를 받아 SDK를 로드합니다.
    - 서버 환경변수 **`KAKAO_MAP_API_KEY`**가 없으면 지도 로드가 실패할 수 있습니다.

## 맘스퀘어(커뮤니티)

### Q. 커뮤니티에서 글을 찾는 방법은?

- **관련 화면**: `/community`
- **키워드**: 검색, 인기 태그, 카테고리, 인기순, 최신순
- **A**:
    - 검색창에 키워드를 입력합니다(제목/내용 기준 필터).
    - 인기 태그(#)를 눌러 검색어를 빠르게 설정할 수 있습니다.
    - 카테고리(자유게시판/육아 꿀팁/Q&A/상담 후기/전문가 칼럼)로 필터링합니다.
    - 인기순/최신순 버튼이 있으나, 실제 정렬 반영은 구현 상태에 따라 다를 수 있습니다.

### Q. 좋아요/북마크는 어떻게 하나요?

- **관련 화면**: `/community`
- **키워드**: 좋아요, 하트, 북마크, 저장
- **A**:
    - 글 카드에서 **하트(좋아요)**, **북마크** 아이콘을 눌러 토글할 수 있습니다.

### Q. 커뮤니티 글쓰기는 어떻게 하나요?

- **관련 화면**: `/community/write`
- **키워드**: 글쓰기, 카테고리, 태그, 이미지 첨부, 등록하기
- **A**:
    - 카테고리를 선택하고(필수), 제목(5자 이상)과 내용(10자 이상)을 입력합니다.
    - 이미지는 **최대 5장**, 태그는 **최대 5개**까지 추가할 수 있습니다.
    - 등록하기를 누르면 `/community`로 이동합니다(현재는 실제 저장 API 없이 이동만 할 수 있습니다).

### Q. 글 상세 보기로 들어가면 404가 떠요.

- **관련 링크**: `/community/{id}`
- **키워드**: 상세 페이지, 404
- **A**:
    - 현재 `Front/app/community/` 아래에 상세 라우트(예: `/community/[id]`)가 없어 **404가 발생할 수 있습니다.**

## 마이페이지/설정

### Q. 마이페이지에서는 뭘 할 수 있나요?

- **관련 화면**: `/mypage`
- **키워드**: 분석 기록, OCR, 아이 관리, 활동, 저장됨
- **A**:
    - 탭으로 구성되어 있으며, 주요 탭은 다음과 같습니다.
        - **개요**: 요약 지표
        - **분석 기록**: 이전 분석 리스트(현재 mock 데이터 기반일 수 있음)
        - **그림일기 OCR**: `/diary-ocr`과 동일 기능
        - **아이 관리**: 등록된 아이 정보(현재 mock 데이터 기반일 수 있음)
        - **활동/저장됨**: 커뮤니티 활동/북마크(현재 mock 데이터 기반일 수 있음)

### Q. 설정에서 무엇을 바꿀 수 있나요?

- **관련 화면**: `/mypage/settings`
- **키워드**: 프로필 설정, 알림 설정, 로그아웃, 계정 삭제
- **A**:
    - 프로필(이름/이메일/연락처) 변경, 알림 토글(이메일/푸시/커뮤니티/마케팅)
    - 보안/구독 관련 메뉴는 현재 **메뉴 형태(UI)**일 수 있습니다.
    - 로그아웃/계정 삭제 버튼이 있으며, 계정 삭제는 확인 다이얼로그가 표시됩니다.
